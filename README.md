# Projeto – Movimentação de Peças de Xadrez

Este projeto implementa a lógica de movimentação das principais peças de xadrez.

## Peças implementadas
- Torre
- Bispo
- Rainha
- Cavalo
- Rei
- Peão

## Funcionalidades
- Validação de movimentos conforme regras básicas
- Verificação de limites do tabuleiro 8x8
- Estrutura orientada a objetos

## Linguagem utilizada
Python

## Como executar
1. Instale o Python
2. Execute no terminal:
python main.py
